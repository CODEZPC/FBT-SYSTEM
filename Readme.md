# 抽号系统配置教程

版本号 2.3.3

## 使用注意事项

为确保最佳体验，建议安装[`extra/FONT.ttf`](extra/FONT.ttf)

> [!IMPORTANT]
> 以下所有事项均在安装了字体的情况下配置

> [!WARNING]
> 错误的格式可能会导致程序非预期地工作！  
> 务必保证格式正确！

> [!TIP]
> 每**3**个半角空格相当于一个中文字符

## 名单

### API

如果要使用CodeAPI上的名单，将[`set/KEY`](set/KEY)文件配置为名单密钥，即可自动获取名单

> [!NOTE]
> 与Github的连接可能不稳定，此时会使用本地名单

例如：

```
KEY-TEST
```

这将会获取到以下名单

```json
{
    "TEST POOL": [
        "测试一",
        "测试二",
        "测试三"
    ]
}
```

### 本地

在[`set/names.json`](set/names.json)中进行配置，格式如下

> [!TIP]
> 如果只希望使用本地名单，改为`NONE`

```json
{
    "名单1": [
        "名字1",
        "名字2"
    ],
    "名单2": [
        "名字1",
        "名字2",
        "名字3"
    ]
}
```

以此类推

## 界面文字及语言

在[`set/lang.json`](set/lang.json)中进行配置，格式如下

```json
{
    "键1": "值1",
    "键2": "值2"
}
```
