# 抽号系统配置教程

版本号 2.3.3

## 名单

### API

如果要使用CodeAPI上的名单，将[`set/KEY`](set/KEY)文件配置为名单密钥，即可自动获取名单

**与Github的连接可能不稳定，此时会使用本地名单**

例如：

```
KEY-TEST
```

这将会获取到以下名单

```json
{
    "TEST POOL": [
        "测试一",
        "测试二",
        "测试三"
    ]
}
```

### 本地

在[`set/names.json`](set/names.json)中进行配置，格式如下

```json
{
    "名单1": [
        "名字1",
        "名字2"
    ],
    "名单2": [
        "名字1",
        "名字2",
        "名字3"
    ]
}
```

以此类推
